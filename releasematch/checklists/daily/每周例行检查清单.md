## 一、四源通道 PoC

```bash
cd releasematch

```


## 二、VPS 容器健康

```bash
# SSH 到当前测试 VPS（IP 见 docs/VPS迁移与部署.md）
docker ps --filter name=jackett --filter name=flaresolverr
```

```
[ ] jackett 容器 Up
[ ] flaresolverr 容器 Up
[ ] 可选：/opt/healthcheck.sh 日志无持续 FAIL
```
