## 三、测速 cron 日志（§4.3）

```bash
# VPS 上
tail -50 /var/log/releasematch/speedtest-cron.log
```

```
[ ] 24h 内至少 1 次 cron 成功完成
[ ] 无连续 Python traceback
[ ] 最近一次 ok/total ≥ 80%（单槽 TTL skip 可接受）
[ ] 无 Jackett / MySQL 持续告警
```
