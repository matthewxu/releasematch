## 一、前置环境

```
[ ] 本机 `.env` 中 `RM_SITE_ORIGIN` 与即将上线的 canonical 域名一致
[ ] `wrangler login` 或 `CLOUDFLARE_API_TOKEN` 可用
[ ] MySQL 可达（`python -m workflow.run db status` → ping.ok）
[ ] Jackett 可达（`python -m workflow.torrent_sources.run status` → reachable）
```

---

## 二、构建与 dist 完整性

```bash
cd releasematch
bash scripts/deploy_cf_pages.sh --prepare-only
```

```
[ ] `deploy_cf_pages.sh --prepare-only` 退出码 0
[ ] `portal/dist/sitemap.xml` 存在且 URL 数符合 D3 规则（≤30 内容页 + Trust + 首页）
[ ] Trust 五页已 rsync 进 dist（about / privacy / dmca / contact 等）
[ ] `portal/static/` 资源已同步进 dist
[ ] `404.html` / `410.html` 已在 dist 根目录
```
