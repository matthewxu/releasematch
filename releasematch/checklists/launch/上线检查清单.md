## 第三篇 · 失败分支速查

| 现象 | 可能原因 | 第一步 |
|------|----------|--------|
| magnet_count &lt; 2 | 源稀缺 / Jackett / 代理 | `torrent_sources.run test --force` · SOCKS |
| 有 magnet 无 Rec | scorer 过滤 | `query page` → rescore → generate |
| 测速 skip no_recommended | 无 Rec | 先修 pipeline |
| dist 无 metadata | 测速后未 generate | 阶段 5 |
| seo_c2 FAIL | dist 过期 / Trust 缺页 | prepare-only → 重跑 |
| genuine_scarcity | 真稀缺 | 登记册，勿无限 `--force` |
