## 〇、垂直边界（先读）

| 垂直 | 本站现状 | SEO 姿态 |
|------|----------|----------|
| **电视剧（真人）** | ✅ `episode.html` + EZTV/Jackett | **主战场 P0** |
| **电影** | ✅ `movie.html` + YTS/Jackett | **主战场 P0** |
| **动漫** | ✅ 走剧集页型 + Nyaa；词表与真人剧不同 | **P1**（源稳定后 index） |
| **日韩真人剧** | ✅ 页型同剧集；双语 title | **P1** |
| **软件 Soft** | ❌ 无页型 / 无 indexer 路由 / 非 TMDB | **域外 — 不做** |
| **游戏 Games** | ❌ 同上 | **域外 — 不做** |
| **音乐 Music** | ❌ 同上 | **域外 — 不做** |

**为什么软件/游戏/音乐不能「顺带覆盖」：**

1. 搜索意图、修饰词、竞品 SERP 与影视完全不同（见 §五）。  
2. 无对应 Recommended / 对版 / 测速 IG → 做成薄页会拖累全站。  
3. 同域混入异质垂直 ≈ Site Reputation / 主题漂移风险（01 文档独立域逻辑的反面）。  
4. 数据源不是 EZTV/YTS，需要整条新 pipeline。

> **本方案原则：** 影视垂直内把词吃透；非影视垂直 **不写进本站 title/desc**，仅保留「若未来分产品线」的词表对照，避免误实施。

## 二、电视剧（TV Episode）— 落地详案

### 2.1 用户怎么搜

| 层级 | 示例 | 本站态度 |
|------|------|----------|
| L3 核心 | `{show} s04e06 download` / `… torrent` | ✅ 主吃 |
| L3 口语 | `{show} season 4 episode 6 torrent` | ✅ 实体已覆盖；desc 用 S04E06 标准形 |
| L4 画质 | `{show} s04e06 1080p web-dl` | ✅ title 已有 res；desc 补 source |
| L4 Group | `{group} {show} s04e06` | ✅ group 非空写入 desc |
| L4 Scene | `… proper` / `repack` / `x264` | ⚠️ title_raw 可见即可 |
| L2 后期 | `{show} torrent` | ⏸ Hub noindex；M8+ |
| 放弃 | `watch {show} online free` | ❌ |

### 2.2 槽位公式

| 槽位 | 公式 |
|------|------|
| **Title（不变）** | `{Show} S{SS}E{EE} Sources — {resolution} \| ReleaseMatch` |
| **Meta description** | `{Show} S{SS}E{EE} torrent sources: Recommended {resolution} {source} ({group}) and {N} matched downloads.` |
| **H1** | `{Show} S{SS}E{EE} — Release-Matched Sources` |
| **Lead（可选）** | `Matched torrent releases for this episode — pick Recommended, not just any magnet.` |
| **H2** | `Recommended Release` / `All torrent sources ({N} matched)` |

**示例：**

```text
Title: Breaking Bad S04E06 Sources — 1080p | ReleaseMatch
Desc:  Breaking Bad S04E06 torrent sources: Recommended 1080p WEB-DL (NTb) and 8 matched downloads.
```

### 2.3 电视剧专属字段 → 词

| 词 | 字段 | 条件 |
|----|------|------|
| SxxExx | season + episode | 必有 |
| 1080p 等 | `recommended.resolution` | 非空 |
| WEB-DL 等 | source / edition | 非空（剧集以 WEB-DL/HDTV 为主） |
| Group | `release_group` | 有 Rec 才写 |
| proper/repack | `title_raw` | 列表展示，不硬塞 desc |

### 2.4 电视剧不要强调的词

- **YTS / YIFY**（电影生态，写进剧集页会误导）  
- **REMUX / BluRay** 作为默认（剧集少见；有则动态出现）  
- **batch**（动漫/合集语境，真人单集页慎用）

## 三、电影（Movie）— 落地详案

### 3.1 用户怎么搜

| 层级 | 示例 | 本站态度 |
|------|------|----------|
| L3 核心 | `{movie} {year} download` / `… torrent` | ✅ 主吃 |
| L4 版本 | `{movie} 1080p bluray` / `remux` / `web-dl` | ✅ All Versions + desc |
| L4 生态词 | `{movie} yts` / `yify` | ✅ 页内诚实 + tier；desc 可选 |
| L4 对比 | `{movie} web-dl vs bluray` | ✅ 多版本分组即 IG |
| L3 magnet | `{movie} {year} magnet` | ⚠️ UI 覆盖即可 |
| 放弃 | `watch {movie} online free` | ❌ |

### 3.2 槽位公式

| 槽位 | 公式 |
|------|------|
| **Title（不变）** | `{Movie} ({Year}) Sources — {resolution} \| ReleaseMatch` |
| **Meta description** | `{Movie} ({Year}) torrent sources: Recommended {resolution} {source} and {N} edition comparisons (WEB-DL / BluRay / REMUX).` |
| **H1** | `{Movie} ({Year}) — Release-Matched Sources` |
| **Lead（可选）** | `Compare edition torrent sources — Recommended picks quality over link count.` |
| **H2** | `Recommended Release` / `All Versions ({N} matched)` |

**示例：**

```text
Title: Inception (2010) Sources — 1080p | ReleaseMatch
Desc:  Inception (2010) torrent sources: Recommended 1080p BluRay and 12 edition comparisons (WEB-DL / BluRay / REMUX).
```

若推荐来自 YTS 系，desc **末尾可加**（诚实）：`YTS-tier listed with group credibility notes.` — 勿写成「best yify crack」。

### 3.3 电影 vs 剧集的关键词差异（实施时勿混用模板）

| 维度 | 电视剧 | 电影 |
|------|--------|------|
| 实体键 | SxxExx | Year |
| 版本叙事 | 单集多 release | **Edition 分组**（WEB-DL/REMUX/BluRay） |
| 高频 L4 | web-dl · scene · group | bluray · remux · yts |
| H2 文案 | All torrent sources | All Versions |
| YTS 词 | 禁止 | 允许 + tier 说明 |
| i18n key | `episode.meta_description` | `movie.meta_description`（**必须分开**） |

## 四、动漫（Anime）— 落地详案（P1）

> 页型仍用 `episode.html`，但 **词表与真人剧分离**；靠路由/类型标记（Nyaa anime / genre Animation）切换文案变体。

### 4.1 用户怎么搜

| 示例 | 态度 |
|------|------|
| `{anime} s01e01 download` / `torrent` | ✅ |
| `{anime} batch 1080p` | ✅ 若页内有 batch 源，需说明单集 vs batch |
| `{anime} {fansub} 1080p` | ✅ Group = fansub |
| `{title} v2` / `dual audio` | ✅ title_raw / 理由 |
| `{日文名} ダウンロード` | P1 双语 |
| `english sub torrent` | ✅ 侧栏字幕 + desc 一句 |

### 4.2 槽位公式（动漫变体）

| 槽位 | 公式 |
|------|------|
| **Title** | 英文主：`{Romaji/EN} S{SS}E{EE} Sources — {res} \| ReleaseMatch`；有日文时可副标题（同一 title 内短拼或 H1 下） |
| **Desc** | `{Title} S{SS}E{EE} anime torrent sources: Recommended {res} ({group}) and {N} matched releases — subs/batch notes on page.` |
| **特有 H2/说明** | 若存在 batch：短注 `Single-episode vs batch releases explained below.` |

### 4.3 动漫专属词 / 勿用词

| 要用 | 勿默认套用真人剧 |
|------|------------------|
| batch · v2 · fansub · dual audio · subs | 默认 YTS、REMUX 叙事 |
| Nyaa group 名 | 仅 EZTV scene 话术 |

**门禁：** Nyaa 源不稳定或 magnet&lt;2 → **noindex**，不抢 anime 词。

## 五、软件 / 游戏 / 音乐 — 域外对照（禁止实施）

### 5.1 为什么要写进文档

避免运营/SEO 把「BT 站通用词」误塞进影视 description（例如 `repack`、`fitgirl`、`flac`），造成：

- 主题漂移、错误流量、高 bounce  
- 政策与 DMCA 风险高于影视 metadata 站  
- 与 Recommended Release / 对版 IG **无法对齐**

### 5.2 各垂直典型词表（仅对照）

#### 软件（Software / Apps）

| 用户常搜 | 说明 | RM |
|----------|------|-----|
| `{app} {version} download` | 正版/破解混杂 | ❌ |
| `{app} portable` / `repack` | 软件包形态 | ❌ |
| `{app} crack` / `keygen` / `activation` | 高风险 | ❌ 严禁 |
| `{app} win11` / `mac dmg` | 平台修饰 | ❌ |

若未来做：**必须独立产品/独立域**，IG = 版本校验、哈希、杀软误报说明等，**不是**测速对版。

#### 游戏（Games）

| 用户常搜 | 说明 | RM |
|----------|------|-----|
| `{game} fitgirl repack` | 压缩包生态 | ❌ |
| `{game} dodi` / `scene` | 发布组 | ❌ |
| `{game} denuvo` / `offline` | 破解语境 | ❌ 高风险 |
| `{game} {year} torrent` | 通用 | ❌ |

若未来做：独立域 + 预装容量/CRC/repack 族谱 IG，与影视 scorer **不能共用**。

#### 音乐（Music）

| 用户常搜 | 说明 | RM |
|----------|------|-----|
| `{album} flac` / `320` | 格式词主导 | ❌ |
| `{artist} discography torrent` | 合集 | ❌ |
| `{album} vinyl rip` | 来源叙事 | ❌ |
| `{track} mp3 download` | 单曲 | ❌ |

若未来做：独立域 + 频谱/cue/log 校验 IG，与 libtorrent「整集测速」无关。

### 5.3 混用红线（本站文案审查）

以下词若出现在 **影视** title/desc 且无数据支撑，视为错误落地：

`crack` · `keygen` · `fitgirl` · `dodi` · `repack`（软件义）· `flac` · `mp3 album` · `portable app` · `windows activate`

> 影视 Scene 的 `REPACK`/`PROPER` 来自 **release 名**，允许在 Sources 表展示；**不要**在电影/剧集 meta 里当软件 repack 营销词写。
