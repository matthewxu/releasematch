关于 ReleaseMatch

ReleaseMatch 是独立的 Release 导航站，帮助用户在某一剧集或电影的众多 BT release 中做出有依据的选择。

我们做什么

  • 聚合公开 indexer 的 torrent 元数据（infohash、标题、做种数等）
  • 通过规则引擎输出 Recommended Release 与对版说明
  • 提供多源交叉验证、Group 信誉与测速摘要

我们不做什么

  • 不托管、不分发视频文件
  • 不提供代下载服务
  • 不保证任何 magnet 的可用性或合法性——用户需自行遵守当地法律

与字幕站的关系

ReleaseMatch 使用独立域名，与字幕主站技术隔离。字幕站仅在单集上下文中提供 一条 出站链接至本站，不进行 sitewide 互链。
