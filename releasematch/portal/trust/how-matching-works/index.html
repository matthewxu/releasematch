对版（Release Matching）如何工作

「对版」指验证某一 release 是否与实际剧集内容完全对应——音轨同步、画质档位、剪辑版本等。多数 magnet 聚合站只列清单；本站在此基础上增加可验证的增量信息。

评分维度

  1. Group 信誉 — 基于历史压制质量的分档（L0~L4）
  2. 跨源交叉验证 — 同一 infohash 在多个数据源是否一致出现
  3. 做种与测速 — seeders 日更 + 连接性/片段测速摘要
  4. 编码规格 — 从 release 名与元数据推断 source、codec、音轨

Recommended Release

对每一槽位（如 Breaking Bad S04E06），引擎对全部候选 release 打分，最高分标记为推荐，并生成嵌入页面的 recommend_reason 文本。

薄页门禁

  • magnet = 0：不建页
  • magnet = 1：noindex
  • magnet ≥ 2 且含 IG 模块：允许 index
